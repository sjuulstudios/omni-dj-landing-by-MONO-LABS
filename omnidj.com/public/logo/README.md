# Logo assets

The site renders the Omni DJ mark from `components/logo/OmniMark.tsx` as inline SVG (parametric 8-circle ring). It does not depend on files here.

Drop these in when you have them, so they can be linked from press kits and OG cards:

- `omni-dj-mark-white.svg` — 8-circle mark, creme/#F5EFE3 on transparent
- `omni-dj-mark-black.svg` — 8-circle mark, black/#0A0A0A on transparent
- `omni-dj-wordmark-white.svg` — "Omni DJ" wordmark, creme on transparent
- `omni-dj-wordmark-black.svg` — wordmark, black on transparent
